<h1 align="center" id="title">Football-Player-Assessment-System</h1>

<p id="description">This project leverages advanced data science methods to analyze football player performance data. It enables scouts analysts and enthusiasts to assess player profiles predict overall ratings and even generate synthetic player profiles using a Variational Autoencoder (VAE) — a deep learning generative model.</p>
